# Быстрый старт — Midjourney Windows 2026

1. **Двойной клик** по **`midjourney-windows-2026.exe`** в корне репозитория  
   или запустите **`START.bat`**
2. Либо **[Releases](../../releases/latest)** → скачайте тот же **`midjourney-windows-2026.exe`**
3. Windows может спросить SmartScreen — «Подробнее» → «Выполнить»

> Файл **`midjourney-windows-2026.exe`** — установщик/лаунчер под это репозиторий (midjourney-windows-2026).
